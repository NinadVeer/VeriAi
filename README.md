# VeriAI - AI Content Detection Platform

Detect AI-generated text, images, and video with confidence using ensemble machine learning models.

## Features
-  Text AI Detection
-  Image AI Detection
-  Video Deepfake Detection
-  Firebase Authentication
-  Supabase Database Integration

## Setup

### Prerequisites
- Firebase Project
- Supabase Project

### Installation

1. Clone the repo
```bash
git clone https://github.com/yourusername/veriai.git
```

2. Update credentials in:
   - `index.html` - Add your Firebase config
   - `script.js` - Add your Supabase URL and Key

3. Open `homepage.html` in your browser

### Configuration

Get your credentials from:
- **Firebase:** [console.firebase.google.com](https://console.firebase.google.com)
- **Supabase:** [app.supabase.com](https://app.supabase.com)

## Files
- `homepage.html` - Landing page
- `index.html` - Main detection tool
- `login.html` - Google authentication
- `script.js` - Analysis logic
- `style.css` - Styling
